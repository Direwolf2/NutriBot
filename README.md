# 🤖 NutriBot — Telegram-бот нутрициолога

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://www.python.org/)
[![aiogram](https://img.shields.io/badge/aiogram-3.x-blue)](https://docs.aiogram.dev/)
[![OpenAI](https://img.shields.io/badge/OpenAI-API-green)](https://openai.com/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

Персонализированный Telegram-бот для расчета калорийности питания и генерации сбалансированного меню на неделю с помощью AI.

## 📋 Описание

**NutriBot** — это интеллектуальный помощник, который:
- 📊 Собирает данные пользователя (пол, возраст, рост, вес, активность, цель)
- 🔥 Рассчитывает базовую калорийность (BMR) по **формуле Миффлина-Сан-Жеора**
- 🎯 Определяет суточную норму калорий с учетом уровня активности и целей
- 📈 Вычисляет индивидуальную норму БЖУ (белки, жиры, углеводы)
- 🍽 Генерирует персонализированное **меню на неделю** через OpenAI API
- 🇷🇺 Учитывает доступность продуктов в России (без экзотики)

Бот использует машинное состояние (FSM) для пошагового сбора данных и интеграцию с **GPT-4** для составления меню.

## 🚀 Быстрый старт

### Требования

- Python 3.9+
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))
- OpenAI API Key (получить на [platform.openai.com](https://platform.openai.com/))

### Установка

1. **Клонируйте репозиторий:**

```bash
git clone https://github.com/ваш-username/nutribot.git
cd nutribot
```

2. **Создайте виртуальное окружение:**

```bash
python -m venv venv
source venv/bin/activate # Для Windows: venv\Scripts\activate
```

3. **Установите зависимости:**

```bash
pip install -r requirements.txt
```

4. **Настройте переменные окружения:**

Создайте файл `.env` в корне проекта:
```bash
TOKEN=your_telegram_bot_token_here
OPENAI_API_KEY=your_openai_api_key_here
```

5. **Запустите бота:**
```bash
python nutri_bot.py
```

## 🎮 Использование

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать заполнение анкеты |
| `/cancel` | Отменить текущее действие |

### Процесс работы

1. **Запустите бота** командой `/start`
2. **Ответьте на вопросы:**
   - Пол (мужской/женский)
   - Возраст (10-120 лет)
   - Вес (30-300 кг)
   - Рост (100-250 см)
   - Уровень активности (минимальная → очень высокая)
   - Цель (похудеть, поддерживать форму, набрать массу)

3. **Получите результаты:**
   - Базовый обмен веществ (BMR)
   - Суточная норма калорий
   - Норма БЖУ (белки/жиры/углеводы)

4. **Дополнительные функции:**
   - 📋 **Мои данные** - посмотреть сохраненную информацию
   - 🔄 **Пересчитать** - заполнить анкету заново
   - 🍽 **Сгенерировать меню** - получить AI-план питания на неделю
   - 🆕 **Новое меню** - создать альтернативный вариант меню

## 🧮 Формулы расчета

### Базовая калорийность (BMR) - формула Миффлина-Сан-Жеора:

**Для мужчин:**
```
BMR = 10 × вес(кг) + 6.25 × рост(см) − 5 × возраст(лет) + 5
```

**Для женщин:**
```
BMR = 10 × вес(кг) + 6.25 × рост(см) − 5 × возраст(лет) − 161
```

### Суточная норма калорий (TDEE):
```
TDEE = BMR × коэффициент_активности × (1 + коэффициент_цели)
```

**Коэффициенты активности:**
- Минимальная (сидячий образ жизни) - 1.2
- Низкая (1-3 тренировки/неделю) - 1.375
- Средняя (3-5 тренировок/неделю) - 1.55
- Высокая (6-7 тренировок/неделю) - 1.725
- Очень высокая (физическая работа + тренировки) - 1.9

**Коэффициенты цели:**
- Похудеть - -20%
- Поддерживать форму - 0%
- Набрать массу - +20%

### Расчет БЖУ:

- **Белки** - 30% калорий (1г = 4 ккал)
- **Жиры** - 25% калорий (1г = 9 ккал)
- **Углеводы** - 45% калорий (1г = 4 ккал)

## 🤖 AI-генерация меню

Бот использует **OpenAI GPT-4** с кастомным промптом, который:
- Создает меню на 7 дней (завтрак, обед, ужин, 2 перекуса)
- Указывает калорийность и БЖУ для каждого блюда
- Использует **только доступные в России продукты** (курица, говядина, рыба, гречка, рис, овощи, фрукты)
- Исключает экзотику (киноа, авокадо, чиа и т.д.)
- Адаптирует меню под цель пользователя

Пример промпта для AI находится в функции `generate_menu()`.

## ⚙️ Настройка

### Изменение модели OpenAI

В функции `generate_menu()` можно заменить модель:  
```
model="gpt-4o" # Более точная модель  
```
или  
```
model="gpt-3.5-turbo" # Быстрее и дешевле  
```

### Настройка БЖУ

Измените пропорции в функции `calculate_macros()`:  
```
protein_calories = calories * 0.30 # 30% белки  
fat_calories = calories * 0.25 # 25% жиры  
carb_calories = calories * 0.45 # 45% углеводы
```

## 🛡️ Обработка ошибок

Бот обрабатывает следующие ошибки OpenAI API:
- `AuthenticationError` - неверный API-ключ
- `RateLimitError` - превышен лимит запросов
- `APIError` - общие ошибки API

Все ошибки логируются через `logging`.
